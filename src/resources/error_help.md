# error help
confused about an error? heres a list of what errors mean (look at the number next to err):
1. grn \[err1\] - common user mistake
2. ylw \[err2\] - issue with user config
3. red \[err3\] - error coming from a C command being used wrong
4. blu \[err4\] - issue within a m7pm command, undefined
5. mag \[err5\] - known issue from a command, gives specifics to an err4 usually
